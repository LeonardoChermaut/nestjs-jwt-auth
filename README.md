# Nest System App Auth - Backend

## Description

This is the API for the login users with JWT implementation.

## Installation

```bash
$ yarn install
or
$ npm install
```

## Running the app

````bash
# development
$ yarn start:dev
or
$ npm start:dev
````

## Test

```bash
# unit tests
$ yarn start test
or
$ npm start test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
````
